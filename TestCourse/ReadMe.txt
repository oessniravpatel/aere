The entry point for this course is an .html file called 'course127775.html'.  
It can be found in the 'course' folder.

The window dimensions you should use to open this course in are 1000px width and 800px height.
An example of how to link to this course can be found in Launch.html

If this course won't be opening in a pop-up window, or if you don't want it to render its own scrollbars, 
you must change 'course127775.html' so "scrolling='no'". (on the last line of this file)
